# 여덟글자 四柱八字 — 사주명리 상담소 모바일 웹

사주전문가의 사주풀이 서비스를 위한 모바일 웹 인트로 페이지입니다.

## 콘셉트

- **겸재 정선의 진경산수화풍**: 한지 바탕, 먹으로 그린 산세(인왕제색도 무드), 산허리 안개, 인주 낙관
- **사람 냄새 나는 카피**: 전문가가 직접 쓴 편지, 손글씨 후기, 삐뚜름한 도장과 종이 질감 등 손맛 위주의 디자인
- 조선 관상감(觀象監)의 전문성·신뢰감을 이야기의 축으로 사용
- 모바일 우선(max-width 480px), 별도 빌드 없이 `index.html` 단일 파일로 동작

## 페이지 구성

| 장 | 내용 |
|---|---|
| 히어로 | 먹산 진경산수 SVG · 낙관 스탬프 |
| 인사 | 전문가가 직접 쓴 편지 (자리 표시용 — 실제 정보로 교체) |
| 유래 由來 | 음양오행 → 이허중 → 서자평 → 조선 관상감 연대기 |
| 원리 原理 | 사주 네 기둥 구조 · 오행 상생 순환도 |
| 맛보기 體感 | 생년월일 + 출생시각 입력 → **명식표(사주판)** 즉석 생성. 월주는 봉인(封) 티저 |
| 풀이 안내 | 평생사주 · 신년운세 · 궁합 · 택일 |
| 후기 | 손글씨 후기 (자리 표시용 — 실제 후기로 교체) |
| 주인장 · CTA | 전문가 프로필 · 풀이 문의 |

## 명식 계산

- **일주**: 기준일 1949-10-01(갑자일) 앵커, 율리우스 적일(JDN) mod 60
- **년주**: 입춘(약 2월 4일) 이전 출생은 전년도로 보정
- **시주**: 오서둔시법 — 일간에 따라 자시의 천간을 정하고 지지 순서대로 진행. 자시 출생은 야자시 논쟁 안내
- **월주**: 절기력이 필요하므로 계산하지 않고 봉인 처리 → 전문가 상담 유도 장치

## 실행

```bash
npx http-server -p 8080
# http://localhost:8080/index.html 을 모바일 뷰포트로 열기
```
