# 사주명리 四柱命理 — 사주풀이 모바일 웹 서비스

사주전문가의 사주풀이 서비스를 위한 모바일 웹 인트로 페이지입니다.

## 콘셉트

- **영화 〈관상〉 포스터 무드**: 먹빛 배경, 한지 질감, 붓글씨 한자, 인주(낙관) 붉은 도장, 금박 포인트
- **조선 관상감(觀象監)의 전문성·신뢰감**을 바탕으로 한 카피와 구성
- 모바일 우선(max-width 480px), 별도 빌드 없이 `index.html` 단일 파일로 동작

## 인트로 구성

| 장 | 내용 |
|---|---|
| 히어로 | 세로쓰기 타이틀 · 낙관 스탬프 애니메이션 |
| 유래 由來 | 음양오행 → 이허중 → 서자평 → 조선 관상감 연대기 |
| 원리 原理 | 사주 네 기둥(四柱) 구조 · 오행 상생 순환도(SVG 애니메이션) |
| 체감 體感 | 생년월일 입력 → 60갑자 일진 계산으로 **일간(日干)·일주·띠** 즉시 감정 |
| 신뢰 信賴 | 관상감 학맥 · 전문가 풀이 소개 |
| CTA | "나의 사주 풀어보기" |

## 일간 계산

기준일 1949-10-01(갑자일)을 앵커로 율리우스 적일(JDN)을 60으로 나눠 일진을 구합니다.
띠(년지)는 입춘(약 2월 4일) 이전 출생 시 전년도로 계산합니다.

## 실행

```bash
npx http-server -p 8080
# http://localhost:8080/index.html 을 모바일 뷰포트로 열기
```
