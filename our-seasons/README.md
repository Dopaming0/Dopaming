# 우리의 계절 — 함께라서 소중한 시간

커플 앱 「우리사이」(`couple/`)와 메멘토 모리 앱 「남은 계절」(`memento-mori/`)을
하나로 합친 앱입니다. 빌드 없이 `index.html` 단일 파일로 동작합니다. (모바일 우선, max-width 480px)

## 합치기 콘셉트

두 앱의 교집합은 **"시간이 유한해서, 함께하는 오늘이 소중하다"** 입니다.

- **뼈대는 우리사이**: 파스텔 코랄의 귀여운 디자인, 문답·기분·추억·기념일·마일스톤 보상 등 매일 쓰는 커플 루프 전부 유지
- **남은 계절은 렌즈로**: 죽음·묘비 같은 어두운 소재는 걷어내고, "함께할 봄이 아직 49번 남았어요" 같은 다정한 언어로 번역해 **계절 탭** 하나에 담음
- 이름 「우리의 계절」 = 우리사이 + 남은 계절

## 새로 생긴 것 — 계절 탭 🌸

| 원본 (남은 계절) | 커플 버전 (계절 탭) |
|---|---|
| 인생 달력 (1칸=1주, 먹색) | **우리의 시간 달력** — 함께한 주는 코랄, 이번 주는 골드, 남은 주는 여백 |
| 횟수의 언어 (남은 봄, 보름달…) | **함께 맞이할 봄 · 나란히 볼 보름달 · 같이 뒹굴 주말 · 여행 · 생일 · 첫눈** |
| 오늘의 물음 (카르페 디엠 30문) | **커플 카르페 디엠 30문** — "오늘이 함께하는 마지막 하루라면?" |
| 후회 예방 체크인 (죽기 전 후회 5가지) | **후회 제로 연습** — 오래된 연인들의 후회 5가지를 오늘의 실천으로 |
| 묘비명 쓰기 | **OUR LAST LINE** — "우리 이야기가 책이 된다면 마지막 문장은?" |
| (로드맵의 버킷리스트) | **우리 버킷리스트** — '언젠가'를 '조만간'으로 바꾸는 목록 |
| 기억의 제단 | 커플 앱 정서와 맞지 않아 제외 (원하면 추가 가능) |

홈 화면에는 매일 바뀌는 티저 카드가 하나 추가됩니다:
"🌸 함께 맞이할 봄 — 앞으로 약 49번 · 그러니까 오늘, 조금 더 다정하게"

## 데이터

- 두 사람의 생일 + 기대수명(기본 83세)은 계절 탭에서 입력, `localStorage`에만 저장
- 저장 키는 우리사이와 같은 `woorisai.v1` — 기존 우리사이 사용자의 기록(문답·추억 등)이 그대로 이어집니다
- 두 기기 동기화(Supabase)·백업/복원도 계절 탭 데이터(생일·후회체크·버킷·마지막 문장)까지 무손실 병합

## v1.2 — 타 커플앱 벤치마킹 기능

| 벤치마크 | 기능 |
|---|---|
| Paired | **커플 퀴즈 (서로 맞히기)** — 매일 1문항. ① 내 답 선택 ② 상대 답 예측, 둘 다 풀면 공개. 누적 **싱크로율** 표시 (질문 탭 + 더보기 통계) |
| 썸원 | **콩닥이 키우기** — 문답·퀴즈·미션 완료가 '밥'이 되어 알 🥚 → 큐피드 하트 💘 7단계 진화. 이름 짓기, 오늘의 먹이 체크, 말풍선 |
| 비트윈 | **사진 추억** — 추억 기록에 사진 첨부. 긴 변 640px·JPEG 62%로 자동 압축(장당 ~30-60KB), 타임라인 썸네일 + 탭하면 크게 보기 |

동기화 최적화: pull은 `updated_at`만 먼저 확인하고 변경이 있을 때만 데이터를 내려받아,
사진이 늘어도 12초 폴링의 통신량이 거의 늘지 않는다.

## v1.3 — 실사용자 리뷰 반영

| 리뷰 | 반영 |
|---|---|
| "커플만의 이야기가 담긴 책으로 만들어주면 좋겠다, 사진도" | **📖 우리의 이야기 책** — 더보기 탭. 표지 → 발자국(통계) → 문답 전체 → 퀴즈 기록 → 사진 추억 → 고마워 모음 → 버킷리스트 → OUR LAST LINE 순서의 책을 만들어 인쇄/PDF 저장 |
| "유한한 게 보여서 슬퍼짐..." | 계절 탭 감정 밸런싱 — "슬퍼지라고 세는 게 아니라 아껴 쓰자고 세는 것" 프레이밍, 이미 함께 보낸 주말 횟수 병기, 히어로에 위로 문구 |
| "버킷리스트 있었으면" (있는 걸 늦게 발견) | 버킷리스트를 횟수의 언어 바로 아래로 승격 — "남은 횟수를 선물로 바꿔요" |

## v1.4 — 오늘의 사랑 한 줄

- 홈 최상단에 매일 바뀌는 사랑 명언 카드 (`quotes.js` 36편 — 저작권 만료 고전 + 자체 문장, 앱·SW 공용)

## 진짜 앱처럼 (PWA)

- `manifest.json` + `sw.js`(네트워크 우선, 오프라인 폴백) + 전용 하트 아이콘(`icon-*.png`, Pillow 스크립트로 생성)
- 홈 화면에 추가하면 전체 화면 standalone 앱으로 열리고, 안드로이드 크롬은 설치 배너·홈 카드의 "설치하기" 버튼 지원
- 브랜드 태그라인: **"인연이 필연이 되는 순간까지"** (온보딩·홈 히어로·더보기 푸터)
